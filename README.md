# CodeMap

O **CodeMap** é uma plataforma web de Tecnologia da Informação que reúne cursos e trilhas de aprendizado para orientar desde iniciantes até gestores.

## 🎯 Objetivo
Facilitar o aprendizado em TI e apoiar o desenvolvimento profissional por meio de um **PDI (Plano de Desenvolvimento Individual)**.

## 👥 Público-Alvo
- Iniciantes em tecnologia  
- Estudantes  
- Profissionais em evolução  
- Gestores  

## 💡 Funcionalidades
- Cursos organizados por área  
- Trilhas de aprendizado  
- Planejamento de carreira (PDI)  

## 🛠️ Tecnologias
- HTML  
- CSS  
- JavaScript
