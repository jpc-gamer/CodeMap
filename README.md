# 💻 CodeMap

O **CodeMap** é um projeto em **HTML e CSS**, desenvolvido no **1º semestre de Fundamentos Web – Mackenzie**.

Um site que reúne **cursos gratuitos e pagos na área de TI**, ajudando no desenvolvimento pessoal e profissional, além de servir como guia para usuários e gestores (PDI).
